Nuevas propiedades de CSS3

Ideas principales

  • Reduce nuestro codigo CSS3, mejorando el diseño y los efectos de movimiento.
  • CSS3 incluye características como esquinas redondeadas, sombras, transformaciones y reposicionamiento.
  • Se deben utilizar prefijos como -moz-, -o- y -webkit- para asegurar que funcione en distintos navegadores.
  • border-radius: Permite crear esquinas redondeadas, podemos definir valores diferentes para cada esquina. Un cuadrado puede convertirse en círculo con 50%.
  • Overflow: hidden; para hacer que una imagen sea recortada por la caja
  • box-shadow: Crea sombras para elementos, especificando desplazamientos y color.
  • text-shadow: Aplica sombras al texto, siendo más mas notorio en textos grandes.
  • La regla @font-face permite cargar y usar fuentes específicas desde el servidor.
  • linear-gradient: Genera degradados lineales y permite especificar colores y puntos de inicio.
  • radial-gradient: Similar a linear-gradient, pero crea degradados radiales con formas de círculo o elipse.
  • RGBA y HSLA: añade un canal alfa para opacidad.
  • border-image: Permite crear bordes personalizados usando imágenes.

Border-radius

Esta propiedad genera esquinas redondeadas para la caja formada por el elemento. Posee dos parámetros diferentes que dan forma a la esquina Cada esquina puede tener su propio valor poniendo cada uno, empieza por la esquina superior izquierda. con diagonal se puede dividir para que la esquina tenga dos valores tomando cada lado de una esquina. El primer parámetro determina la curvatura.

Box-shadow

Crea sombras para la caja formada por el elemento.

El primer valor es para el desplazamiento vertical, horizontal, difuminación y color, sucesivamente.

Text-shadow

Da sombras al texto, es mejor con texto muy grande para que sea notorio

Admite cuatro valores al igual que box-shadow menos inset.

@font-face

Esta regla nos permite cargar y usar cualquier Fuente que necesitemos. Primero, debemos declarar la Fuente, proveer un nombre con la propiedad font-family y especificar el archivo con src. Luego podremos asignar la fuente a cualquier elemento del documento

Linear-gradient

Puede ser aplicada a background o background-image para generar un gradiente lineal. Es bueno agregar un color de fondo por si acaso el navegador no soporta los degradados. Los atributos indican el punto inicial y los colores para crear el gradiente. El primer valor puede ser especificado en pixeles, en porcentaje o usando las palabras clave top, bottom, left y right. El punto de inicio puede ser reemplazado por un ángulo para proveer una dirección específica para el gradiente.

Radial-gradient

Prefijo de vendedor -o- Esta función puede ser aplicada a las propiedades background o brackground-image para generar un gradiente radial. La posición de inicio es el origen y puede ser declarado en pixeles, porcentaje o como una combinación de las palabras clave center, top, bottom, left y right. Existen dos valores para a forma: circle y allipse, y puntos de terminación pueden ser declarada para cada color indicando la posición donde la transición comienza.

RGBA

Se agrego la A, canal alfa sirve para la opacidad. También se encuentra el hexadecimal con alfa y hsla. Esta función es una mejora de rgb con un canal alfa que especifica la opacidad de un color.

Toma cuatro valores

El color rojo, verde, azul y opacidad.

HSLA

Significa tono, saturación y luminosidad. Un valor de color HSL se especifica con tono,grado en la rueda de color; el parámetro alfa define la opacidad, que es un numero entre 0.0

Saturación: 100% es el color completo

Luminosidad: 0% es oscuro y 100% es blanco

Apuntes de la clase

Outiline

Esta propiedad fue mejorada con la incorporación de otra propiedad llamada outlint-offset. Ambas propiedades combinadas generan un segundo borde alejado del borde original del elemento

Border image

Esta propiedad crea un borde con una imagen personalizada. Necesita que border sea declaro previamente con las propiedades border o border-with, y toma al menos tres parámetros: la url de la imagen, el tamaño de las piezas que serán tomadas de la imagen para construir el borde y una palabra clave que especifica como esas piezas serán ubicadas alrededor del elemento

Transform

Esta propiedad modifica la forma de un elemento. Utiliza cuatro funciones básicas:

Scale (escalar), rotate (rotar), skew (inclinar), translate (trasladar o mover): La función scale recibe solo un parámetro. Un valor negativo invierte el elemento, valores entre 0 y 1 reducen el elemento y valores mayores que 1 expanden el elemento. La función rotate usa solo un parámetro expresado en grados para rotar el elemento. La función skew recibe dos valores, también en grados para la transformación horizontal y vertical. La función translate mueve el objeto tanto pixeles como sean especificados por sus parámetros.

Resumen

CSS3 introduce mejoras significativas en el diseño web, permitiendo estandarizar funciones que antes requerían JavaScript. Border-radius: Permite crear bordes curvados en los elementos.

Con box-shadow y text-shadow, se pueden agregar sombras a elementos y textos, mejorando la estética. La regla @font-face facilita la carga de fuentes desde el servidor. Las funciones linear-gradient y radial-gradient permiten crear degradados en fondos de manera sencilla.

A través de RGBA y HSLA, se pueden especificar colores con canales alfa para controlar la transparencia. Se puede personalizar la apariencia de los bordes con outline y border-image.

La propiedad transform permite modificar la forma de los elementos mediante escalado, rotación, inclinación y movimiento.
