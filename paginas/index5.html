Referencias avanzadas

Ideas principales

  • Id: Identifica de manera única a un elemento en el documento. No puede ser duplicado, tiene ser único.
  • Class: permite asignar el mismo nombre a múltiples elementos que comparten un diseño.
  • Elementos: Seleccionan todos los elementos de un tipo específico.
  • :nth-child(): Selecciona un hijo específico.
  • :first-child y :last-child: Seleccionan el primer y el último hijo, respectivamente.
  • :only-child: Afecta un elemento SOLO si es el único hijo
  • :not(): Aplica estilos a todos los elementos excluyendo los que coinciden con la referencia dentro de los paréntesis.
  • [atributo="valor"] [atributo^="valor"] [atributo$="valor"][atributo*="valor"]: elementos con un atributo específico, elementos con atributo que comienza con un valor específico, elementos con atributo que termina con un valor específico, elementos con atributo que contiene una cadena específica, respectivamente.
  • hijos directos (>): elementos que son hijos inmediatos de un padre.
  • hermanos adyacentes (+): el hermano inmediato que sigue a otro elemento.
  • hermanos generales (~): todos los hermanos de un elemento, inmediatos o no.

Apuntes de la clase

Atributo id

El atributo id es como un nombre que identifica al elemento. El valor de este atributo no puede ser duplicado. Este nombre debe ser único en todo el documento.

Atributo class

Es más flexible y puede ser asignado a varios elementos HTML en el documento que comparten un diseño similar

Elementos

Un selector de elementos en CSS es una regla que selecciona todos los elementos de un tipo especifico en el documento HTML. Es el selector más básico y se refiere directamente al nombre de una etiqueta

Pseudoclases - :nth-child()

La pseudoclase nth-child() nos permite encontrar un hijo especifico

:nth-chil(odd) :nth-child(even)

Las palabras clave odd (impar) y even (par) se utilizan para seleccionar elementos para seleccionar elementos con base en su posición en la secuencia de hijos, específicamente para alternar entre elementos impares y pares

:first-child

Referencia solo el primer hijo. Estas pseudoclases en particular no requieres palabras clave o parámetro

:last-child

Referencia solo al último hijo. Estas pseudoclases en particular no requieres palabras clave o parámetro.

Only-child

Afecta un elemento siempre y cuando sea el único hijo disponible. Estas pseudoclases en particular no requieres palabras clave o parámetro

:not()

Nos permite declarar una excepción. Los estilos en la regla creada con esta pseudoclase serán asignados a todo elemento excepto aquellos incluidos en la referencia entre paréntesis En lugar de la palabra clave de un elemento podemos usar cualquier otra referencia que deseemos.

Selectores de atributo

Un selector de atributo en CSS selecciona elementos en función de la presencia o el. Valor de uno de sus atributos. Esto permite aplicar estilos a los elementos que tienen un atributo especifico, o cuyo atributo tiene un valor determinado.

[atributo ^=”inicio”]

Una regla con el selector (^=) selecciona elementos cuyo atributo comienza con un valor especifico

[atributo$=”final”]

Una regla con el selector ($=) selecciona elementos cuyo atributo termina con un valor especifico

[atributo*=””]

Una regla con el selector (*=) selecciona elementos cuyo atributo contiene una cadena especifica en cualquier posición

Selectores de relación

Los selectores de relación en CSS permiten seleccionar elementos basados en su relación con otros elementos dentro del documento. Estos selectores son útiles para aplicar estilos a elementos que tienen un vínculo estructural con otros elementos, como ancestros, padres, hermanos o descendientes.

Selector de hijos directos (>)

Selecciona los elementos que son hijos inmediatos de un elemento padre. nota: Por lo cual no aplica a los descendientes

Selector de hermanos adyacentes (+)

Selecciona un elemento que es hermano inmediato, es decir, está justo después de otro elemento.

Selector de hermanos generales (~)

Selecciona a los elementos hermanos de un elemento anterior, sean hermanos inmediatos o no.

Resumen

Permite una selección más precisa de elementos a través de varios atributos y selectores

id: Identifica de manera única un elemento en el documento, por lo cual, no puede repetirse. class: Permite asignar el mismo nombre a varios elementos con un diseño similar.

Facilitan la selección de elementos basados en su posición o estado, como :nth-child(), :first-child, :last-child, :only-child, y :not().

Los selectores de atributo, nos permiten seleccionar elementos según la presencia o el valor de sus atributos, utilizando operadores como ^=, $=, y *=.

Los selectores de relación, sirve para seleccionar elementos segun su relacion con otros. Hijos directos (>), hermanos adyacentes (+), hermanos generales (~).
