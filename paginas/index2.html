Propiedad display

Ideas principales

  • Da el tipo de caja de un elemento, tiene valores predeterminado como block o inline
  • Block: cominezan en una nueva linea y ocupan todo el ancho
  • Se usa en los div, h1-h6, p, form, etc.
  • Inline: no comineza una linea nueva y ocupa el ancho necesario
  • Se usa en span, a, img
  • Inline-block: Permite definir ancho y altura. Respeta margin y padding. No añade un salto de línea, permitiendo alinearse junto a otros elementos.
  • None: Utilizado comúnmente con JavaScript para ocultar elementos. No genera caja; el elemento desaparece y no ocupa espacio.
  • Table: Asigna comportamiento de tabla a los elementos anidados.
  • table-row actúa como una fila (tr).
  • table-cell actúa como una celda (td).

Apuntes de la clase

Establece el tipo de caja que genera cada elemento, que cada uno tiene un valor predeterminado según el tipo de elemento que sea. El valor predeterminado block o inline.

Block

Siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como sea posible).

Elementos con display block

Div, h1-h6, p, form, header, footer, section

Inline

No comienza en una nueva línea y solo ocupa el ancho que sea necesario.

Elementos en línea:

Span, a, img

Inline-block

Permite establecer un ancho y una altura en el elemento. Se respetan los margin y padding, con inline no lo son. No agregan un salto de línea después del elemento, por lo que puede alinearse junto a otros elementos.

None

Se usa comúnmente con JavaScript para ocultar y mostrar un elemento cuando el usuario realiza alguna acción como pulsar un botón. No genera ninguna caja. El elemento desaparece por completo de la página y no ocupa sitio, por lo que los elementos adyacentes ocupan su lugar. Si se utiliza la propiedad sobre un elemento, todos sus descendientes También desaparecen por completo de la pagina.

Table

Asigna un comportamiento de tabla a los elementos que haya dentro (anidados a la etiqueta donde estas colocando el display: table;). Pueden usar otra serie de valores display que corresponden con distintas etiquetas HTML que usas en las tablas.

Display: table-row; actúa como una fila, etiqueta “tr”.

Display: table-cell; etiquetas que trabajan como una celda, etiqueta “td”.

Resumen

La propiedad display nos ayuda a determinar cómo se visualizan los elementos en una página web. Los valores principales son:

block: Los elementos comienzan en una nueva línea y ocupan todo el ancho disponible, por ejemplo: div, h1, p.

inline: Los elementos no inician en una nueva línea y solo ocupan el ancho necesario, por ejemplo: span, a, img.

inline-block: Permite establecer ancho y altura, respeta margin y padding, y no agrega saltos de línea.

none: Oculta elementos de la página, haciendo que no ocupen espacio.

table: Permite que los elementos anidados tengan un comportamiento similar al de tablas, con valores como table-row y table-cell.
